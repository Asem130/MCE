<h1 style="font-size: 32px; text-align: center; color: #4CAF50;">🌟 MCEKW - Currency exchange App 🌟</h1>

<p style="font-size: 18px; text-align: justify;">MCEKW is a powerful currency conversion app that enables you to easily convert Kuwaiti Dinar to multiple currencies with real-time exchange rates. 💱 The app also tracks currency trends through an interactive chart 📊, provides secure digital wallet storage 💳, and allows seamless payments with TAB integration 💳. Additionally, MCEKW offers fast and reliable home money delivery services 🏠💵, ensuring a hassle-free experience.</p>

<h2 style="font-size: 28px; color: #4CAF50;">✨ Features ✨</h2>
<ul>
  <li><span style="font-size: 20px;">💰 Convert Kuwaiti Dinar to multiple currencies with live exchange rates.</span></li>
  <li><span style="font-size: 20px;">📈 Track currency trends with an interactive chart.</span></li>
  <li><span style="font-size: 20px;">🔒 Store money securely in digital wallets.</span></li>
  <li><span style="font-size: 20px;">💳 Seamless payments with TAB integration.</span></li>
  <li><span style="font-size: 20px;">🚚 Fast and reliable home money delivery.</span></li>
  <li><span style="font-size: 20px;">🔑 Use Face ID for app login.</span></li>
  <li><span style="font-size: 20px;">🔔 Send notifications to users for important updates.</span></li>
  <li><span style="font-size: 20px;">📄 Pagination for organized data display.</span></li>
  <li><span style="font-size: 20px;">🌍 Language change support based on user preference.</span></li>
  <li><span style="font-size: 20px;">🖥️ User-friendly and flexible interface.</span></li>
</ul>
